## 프로젝트 개요

### 기술 스택
- **백엔드**: FastAPI (Python)
- **프론트엔드**: Next.js 13.5.6, React 18.3.1, TypeScript 5.9.2
- **데이터베이스**: MariaDB (최신 버전), Redis (캐시)
- **인프라**: Docker Compose, Naver NCloud
- **UI**: Radix UI, Tailwind CSS, shadcn/ui 스타일

### 개발 환경
- Docker Compose를 이용한 컨테이너 관리
- 로컬 개발 환경 구축
- CI/CD: GitHub Actions

### 개발 순서
**프론트엔드 우선 개발 방식**: 프론트엔드를 먼저 개발하고 Mock API/더미 데이터를 사용하여 UI/UX를 완성한 후, 완성된 프론트엔드 요구사항을 기반으로 백엔드 API를 개발합니다.
